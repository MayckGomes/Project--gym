# PROJECT GYM

esse foi um projeto realizado para aprendizado de novas ferramentas do desenvolvimento android.

## Funcionalidades :

Com esse app, Você pode organizar sua rotina de treino, com interfaces intuitivas e fáceis de entender.

## Elementos usados:

- button
- text
- box
- navBar
- lazyColumn
- roomDB
- coroutines
- checkbox
